## Objetivo

Estos documentos existen para que una IA que trabaje en este repo tenga
contexto suficiente de qué es (y qué no es) esta app antes de tocar código.
Es un shell nativo sin lógica propia — la decisión completa de por qué
existe vive en `sazono-ui/docs/13-app-mesero-capacitor.md`, y las reglas de
producto que renderiza viven en `sazono-ui/docs/01-product-context-ui.md` y
`sazono-ui/docs/06-mesero-y-cocina.md`.
